# SAM2 세그먼테이션 라벨링 툴

##  **툴 목적**
이미지에서 객체를 자동으로 인식하고 정확한 폴리곤 라벨을 생성하는 AI 기반 라벨링 도구입니다.
- **SAM2 (Segment Anything Model 2)** 를 활용한 고정밀 자동 세그먼테이션
- **수동 편집 기능**으로 완벽한 라벨 생성
- **다양한 학습 포맷** 자동 변환 (YOLO, SegMAN, JSON)

##  **핵심 기능**

### **자동 세그먼테이션**
- **SAM2 AI 모델**: 클릭 한 번으로 정확한 객체 인식
- **스마트 폴리곤**: 복잡한 형태도 깔끔한 다각형으로 변환
- **홀 채우기**: 내부 빈 공간 자동 처리

### **수동 편집**
- **3가지 모드**: SAM 자동 / 그리기 / 지우개
- **실시간 미리보기**: M(컬러), N(흑백), S(단순화)
- **Undo 기능**: 실수 즉시 되돌리기

### **고급 기능**
- **Split Mode**: 좌우 분리 라벨링 (겹치는 객체 처리)
- **Polygon Simplify**: 외곽선 직선화 (0~5단계)
- **배경 시각화**: 일반/엣지/대비 모드

### **출력 포맷**
- **YOLO Segmentation**: `*.txt` 정규화 좌표
- **SegMAN**: `*.png` 마스크 이미지
- **Simple JSON**: 원본 픽셀 좌표
- **시각화**: 결과 확인용 이미지

##  **사용법 시나리오**

### **시나리오 1: 기본 라벨링**
```bash
# GUI 실행
python src/label_tool_gui.py
```

1. ** 입력 폴더**: 라벨링할 이미지 폴더 선택
2. ** 출력 폴더**: 결과 저장할 상위 폴더 선택  
3. ** 폴더 이름**: `labels_20250117_1430` (자동 생성)
4. ** Start Labeling**: 라벨링 시작

**라벨링 과정:**
- **좌클릭**: 포함할 영역 선택
- **우클릭**: 제외할 영역 선택  
- **SPACE**: 저장하고 다음 이미지
- **백틱(`)**: 현재 이미지 스킵
- **ESC**: 종료

### **시나리오 2: 정밀 라벨링**
```bash
# 복잡한 객체나 겹치는 객체 처리
```

1. **모드 전환**:
   - `1`: SAM 자동 모드 (기본)
   - `2`: 그리기 모드 (세부 추가)
   - `3`: 지우개 모드 (불필요 부분 제거)

2. **Split Mode** (좌우 분리):
   - `Q`: 왼쪽 영역만
   - `W`: 오른쪽 영역만  
   - `E`: 전체 영역

3. **미리보기**:
   - `M`: 컬러 마스크 확인
   - `N`: 흑백 마스크 확인
   - `S`: 폴리곤 단순화 조절

### **시나리오 3: 작업 이어하기**
```bash
# 중단된 작업 재개
```

1. **🔄 작업 이어하기** 버튼 클릭
2. **progress_log.json** 파일 선택
3. 자동으로 입력/출력 폴더 설정
4. 이미 완료된 이미지는 자동 스킵

### **시나리오 4: CLI 모드** (고급)
```bash
# 명령줄에서 직접 실행
python src/label_tool.py
```

- OpenCV 윈도우에서 직접 라벨링
- 서버 환경이나 자동화에 적합

##  **키보드 단축키**

| 키 | 기능 | 설명 |
|---|---|---|
| **1/2/3** | 모드 전환 | SAM/그리기/지우개 |
| **Q/W/E** | Split Mode | 왼쪽/오른쪽/전체 |
| **M/N/S** | 미리보기 | 컬러/흑백/단순화 |

| **U/Ctrl+Z** | Undo | 마지막 작업 취소 |
| **V** | 배경 변경 | 일반/엣지/대비 |
| **C** | 색상 변경 | 오버레이 색상 |
| **F** | 홀 채우기 | ON/OFF 토글 |
| **R** | 리셋 | 현재 이미지 초기화 |

##  **사용 팁**

### **효율적인 라벨링**
- **SAM 모드 우선**: 대부분 자동으로 정확하게 인식
- **좌우 분리**: 겹치는 객체는 Q/W 키로 분리 처리
- **미리보기 활용**: M/N 키로 결과 미리 확인

### **품질 향상**
- **Polygon Simplify**: 울퉁불퉁한 경계선은 S 키로 조절
- **홀 채우기**: 격자나 구멍이 있는 객체는 F 키로 ON/OFF
- **수동 편집**: 세부 조정은 그리기/지우개 모드 활용

### **문제 해결**
- **느린 로딩**: 첫 실행 시 모델 다운로드 (5~10초)
- **메모리 부족**: 16GB RAM 권장, GPU 메모리 부족 시 재시작
- **이미지 사라짐**: 입력/출력 폴더가 같으면 복사 안 됨
